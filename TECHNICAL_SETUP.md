# Technical Setup for Spaceship Horror Demo

## Unity Configuration
1. **Unity Version**: Make sure to use Unity 2020.3 or later for optimal performance and compatibility.
2. **Graphics API**: Use Direct3D 11 for Windows build and Metal for iOS devices.
3. **Quality Settings**: Set the quality to High for better visual fidelity. Adjust the shadows and anti-aliasing based on target platforms.

## Technical Requirements
1. **Minimum System Requirements**:
   - OS: Windows 7/8/10, macOS 10.12 or later
   - Processor: Intel Core i5 or equivalent
   - Memory: 8 GB RAM
   - Graphics: NVIDIA GeForce GTX 660 or equivalent
   - DirectX: Version 11

2. **Recommended System Requirements**:
   - OS: Windows 10, macOS 10.15 or later
   - Processor: Intel Core i7 or equivalent
   - Memory: 16 GB RAM
   - Graphics: NVIDIA GeForce GTX 970 or equivalent
   - DirectX: Version 12

## Asset Recommendations
1. **Textures**: Use 2048x2048 resolution textures for 3D models and 1024x1024 for UI elements.
2. **Sound Effects**: Include immersive sound effects for an engaging experience. Recommended samples for sci-fi atmosphere.
3. **Models**: Utilize optimized 3D models with proper LOD (Level of Detail) settings for performance. Look for models on Unity Asset Store or other reputable sites.

4. **Animations**: Consider using pre-built animations for character actions from Mixamo or similar resources.

Make sure to keep this file updated as the project evolves.
